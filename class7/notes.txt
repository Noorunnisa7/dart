Arrays in Dart (List, Map, Set)
-Dart does not have a separate array type
instead, it uses List, Map, and Set as its main collection types.

1. List (Equivalent to Arrays)
-A List is an ordered collection of items.
Indexed starting from 0.
 List<dataType> a = [values...];

3. Set (Unique items, no duplicates)
-A Set is an unordered collection of unique elements.
